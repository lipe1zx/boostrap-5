Cores de Fundo

As Classes de fundo alteram a cor de fundo do elemento, criando blocos de cor que podem destacar seções ou criar contrastes visuais

.bg-primary (Azul)

Usado para destacar seções importantes como cabeçalhos ou rodapes.

.bg-secondary (cinza)

Ideal para areas de conteudo secundario ou para criar constraste com elementos principais.

.bg-success (Verde)

Indica áreas de sucesso, como mensagens de confirmação ou seções de feedback positivo.

.bg-danger (Vermelho)

Destaca áreas de erro ou perigo, como mensagens de alerta ou seções críticas.

.bg-warning (Amarelo)

Usado para chamar atenção para áreas que requerem cuidado, como avisos ou seções de atenção.

.bg-info (Ciano)

Ideal para destacar informações de suporte ou seções de dicas e notificações.

.bg-dark (Preto/Cinza Escuro)

Usado para criar contraste forte, especialmente em seções de rodapés ou áreas de destaque.

.bg-light

Usado para criar áreas de fundo claras e suaves, ideal para seções de conteúdo ou para criar contraste com elementos mais escuros.

Cores sutis e enfase

Estas são as cores mais modernas do bootstrap. elas usam tons pasteis para o fundo e cores fortes para o texto, criando um contraste perfeito sem "agredir" os olhos

.bg-primary-subtle + .text-primary-emphasis

combinação de fundo suave com texto azul forte, ideal para seções que precisam de destaque sem ser agressivas

.bg-secondary-subtle + .text-secondary-emphasis

Fundo cinza claro com texto cinza escuro, perfeito para areas de conteudo secundario ou informações de apoio

.bg-danger-subtle + .text-danger-emphasis

Fundo vermelho claro com texto vermelho forte, perfeito para alertas de erro ou seçoes criticas que precisam de destaque

.bg-warning-subtle + .text-warning-emphasis

Fundo amarelo claro com texto amarelo forte, ideal para avisos que requerem atenção sem ser agressivos

.bg-info-subtle + .text-info-emphasis

Fundo cioano claro com texto ciano forte, perfeito para destacar informações de suporte ou seções de dicas e notificações

.bg-dark-subtle + .text-dark-emphasis

Fundo cinza escuro claro com texto preto forte, ideal para criar contraste em seções de rodapes ou areas de destaque sem ser agressivo
